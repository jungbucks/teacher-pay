# 교사 수당 계산기 — UI 리프레시 설계 (2026-07-19)

> v1.1의 "연속용지 급여대장(greenbar)" 모티프를 **토스/애플식 미니멀 모던 UI**로 리팩토링한다.
> 계산 로직·데이터·규정(RULES_VERIFIED=true)·검증 체계는 **그대로 유지**. 바꾸는 것은 오직 표현층(HTML 구조 + CSS + 폰트).
> 세계관 전환: "장부 명세서" → **"차분한 명세 위에 근속의 리듬을 한 곳에서 보여주는 계산기"**.

## 0. 확정 사항 (브레인스토밍)

- 방향: 토스/애플식 미니멀. 4대 원칙(아래 §2) 반영.
- 초록 농도: **액센트 + 상단 브랜드 밴드**. 면(배경·카드·테두리·일반 텍스트)은 무채색, 초록은 액센트로만.
- 기술: **바닐라 CSS**(Tailwind 미채택 — 빌드·CDN 없이 `file://` 더블클릭 원칙 유지). Tailwind의 *룩*만 바닐라로 재현.
- 폰트: **Paperlogy(디스플레이·큰 숫자·라벨) + Pretendard(본문·주석)** 페어링. Paperlogy는 OFL이라 woff2 셀프호스팅.
- 시그니처: **월별 수당 지급 타임라인**(12개월 리본) 채택.

## 1. 목적 (불변)

호봉·근무연수만 넣으면 정근수당·정근수당 가산금·명절휴가비가 즉시 뜨는 공개 계산기. 단일 임무·입력↔결과 실시간·모바일 우선은 그대로. 이번 작업의 목표는 **시각적 노이즈 제거 + 정보 위계 강화 + 새 폰트 정체성**이다.

## 2. 4대 개선 원칙 (사용자 브리핑)

1. **장식·시각 노이즈 제거**
   - 우측 명세서의 쿠폰형 **천공 점선(perf) 테두리 삭제** → 은은한 그림자 카드.
   - **greenbar 줄무늬(repeating-linear-gradient) 삭제** → 순수 흰 카드.
   - 우측 하단 **빨간 "참고" 직인 그래픽 완전 삭제**(참고용 고지는 푸터 텍스트로 존재 → 정보 손실 없음). PNG 내보내기의 직인도 삭제.
   - 초록 밴드 헤더(`.slip-band`) → 무채색 카드 헤더(제목 먹색 + 중립 칩).
2. **레이아웃·균형**
   - 좌(계산 조건)·우(급여 명세) 카드 **동일 높이(stretch)·동일 내부 패딩**.
   - 카드 아래 떠 있던 [결과 복사]·[이미지로 저장] 버튼 → **두 카드 아래 화면 하단 중앙 정렬 한 줄**(토스식 하단 액션 바). 카드 안에 넣지 않는 이유: 우측 카드만 길어져 균형이 깨지므로.
3. **색상·시선 유도**
   - 배경·일반 카드: 화이트/오프화이트/그레이 모노톤.
   - 초록 포인트는 **① 연간 합계 금액·블록 ② 주요 액션 버튼 ③ 슬라이더/활성 호봉 셀 ④ 상단 브랜드 밴드 마크**에만.
4. **타이포 위계 강화**
   - 숫자(호봉·금액): 굵고 큼(800). 부연·주석: 작고 담백(`--ink-faint`, 0.82rem).

## 3. 토큰 시스템

### 3-1. Color (모노톤 캔버스 + 단일 초록 액센트)

| 토큰 | 값 | 쓰임 |
|---|---|---|
| `--canvas` | `#f5f6f5` | 페이지 배경(쿨그레이-오프화이트) |
| `--surface` | `#ffffff` | 카드 |
| `--ink` | `#1a1d1b` | 큰 숫자·제목(초록 기 도는 먹) |
| `--ink-soft` | `#5b6560` | 본문 |
| `--ink-faint` | `#8a938d` | 캡션·부연(text-gray-500 상당) |
| `--line` | `#ecefec` | 헤어라인 구분선·카드 테두리 |
| `--brand` | `#17553b` | **액센트 전용**(기존 브랜드 딥그린 유지) |
| `--brand-strong` | `#0f3f2b` | 버튼 hover/pressed |
| `--brand-tint` | `#eef4f0` | 연간 합계 블록 배경·활성 상태 |

- `--seal`(레드) 토큰과 관련 그래픽 **제거**.

### 3-2. Type — Paperlogy × Pretendard 페어링

- **디스플레이·큰 금액·라벨: Paperlogy**(700/800, 필요시 900). 영문·숫자가 Montserrat 기반이라 큰 금액의 기하학적 인상이 주인공이 됨.
- **본문·주석·캡션: Pretendard → "Malgun Gothic" → system-ui 폴백.**
- 폰트 스택:
  - `--font-display: "Paperlogy", "Pretendard", "Malgun Gothic", system-ui, sans-serif;`
  - `--font-body: "Pretendard", "Malgun Gothic", system-ui, sans-serif;`
- **모노스페이스(Consolas) 은퇴.** 금액은 더 이상 모노가 아니라 Paperlogy + `font-variant-numeric: tabular-nums`.
- 타입 스케일(기준값, 반응형 축소):
  - 연간 합계 금액: Paperlogy 800 · 2.4rem
  - 개별 수당 금액: Paperlogy 700 · 1.4rem
  - 헤드라인(히어로): Paperlogy 800 · 2.0rem
  - 라벨(근무연수/호봉): Paperlogy 700 · 1.05rem
  - 본문: Pretendard 500 · 0.95rem
  - 캡션/주석: Pretendard 400 · 0.82rem · `--ink-faint`
- 본문 15px 이상 유지(CLAUDE.md 규칙 5).

### 3-3. Layout

```
┌───────────────────────────────────────────────┐
│ ● 2026 교원 봉급표 기준 · 참고용   ← 얇은 브랜드 밴드(초록 점)  │
│                                               │
│  근무연수와 호봉, 둘만.        ← Paperlogy 먹색 대형 헤드라인  │
│  정근수당·가산금·명절휴가비를 한눈에.               │
├─────────────────────┬─────────────────────────┤
│  계산 조건            │  급여 명세   [2026·세전]   │  ← 동일 높이·패딩
│                     │                         │
│  근무연수     10      │  정근수당 50%   1,740,500원 │
│  ──●────── 슬라이더   │  가산금        60,000원/월 │
│                     │  명절휴가비    2,088,600원 │
│  호봉         20      │  ─────────────────────  │
│  ────●─── 슬라이더    │  ┌ 연간 합계 8,378,200원 ┐│ ← brand-tint 블록
│  월봉급 3,481,000원    │  └───────(초록)────────┘│
│                     │  [월별 지급 타임라인 리본]    │ ← 시그니처(§4)
└─────────────────────┴─────────────────────────┘
        [ 결과 복사 ]   [ 이미지로 저장 ]      ← 하단 중앙 액션 바
      ▸ 2026 전체 호봉표          ▸ 이게 무슨 수당인가요
                    (푸터: 참고용 고지·출처)
```

- 데스크톱(≥860px): 2열 그리드 `21rem 1fr`, `align-items: stretch`로 동일 높이. 모바일: 1열 스택.
- 두 카드 동일 내부 패딩(예: 1.5rem). 스텝(±) 버튼은 유지하되 미니멀 스타일(중립 테두리, hover 시 브랜드).
- 카드 = `--surface` + `--line` 1px + 은은한 그림자(`0 1px 3px rgba(20,30,25,.06), 0 8px 24px rgba(20,30,25,.05)`), `border-radius: 16px`.
- 하단 액션 바: 주요 = 초록 채움("이미지로 저장"), 보조 = 고스트("결과 복사"). 중앙 정렬.
- 호봉표(`details`)·용어 설명(`explainer`)·푸터: 모노톤으로 재스타일, 구조·문구 유지.

### 3-4. Signature — 월별 수당 지급 타임라인

우측 명세 카드 하단(연간 합계 아래)에 **12개월 가로 리본**. 이 계산기의 정보(이미 주석에 있는 "1·7월 / 매월 / 설·추석")를 시각화하여, 숫자만 나열하는 흔한 계산기와 차별화한다. 구조가 곧 정보(달이 실제 순서를 가짐).

- 12칸(1월~12월) 얇은 트랙. 각 달에 그 달 지급되는 수당을 초록 마커로 표기:
  - **정근수당**: 1월·7월
  - **정근수당 가산금**: 매월(12개월 전부, 옅은 상시 표시)
  - **명절휴가비**: 설·추석 → **간이 규칙으로 2월·9월에 고정 표기**하고, "설·추석은 해마다 달라져 대표 월로 표시"임을 캡션에 명시(음력이라 정확 월은 매년 변동 → 숫자 지어내지 않기 원칙 준수, 근사임을 밝힘).
- 마커에 호버/포커스 시 그 달 금액 요약(선택적, 접근성 위해 `title`/`aria-label`).
- 초록은 여기서도 액센트로만: 가산금 상시 = 옅은 tint, 정근·명절 이벤트 달 = 진한 브랜드 점.
- 리본은 **표시 전용**(계산 결과의 재현). 새 데이터·새 계산 없음 → 검증 부담 최소.
- reduced-motion 존중(등장 애니메이션 있으면 비활성).

## 4. 컴포넌트별 변경 요약 (표현층만)

| 요소 | 현재(v1.1) | 리프레시 |
|---|---|---|
| masthead | 큰 초록 그라디언트 헤더 | 얇은 브랜드 밴드 + 흰 배경 먹색 헤드라인 |
| 카드 테두리 | 천공 점선(perf) | 삭제 → 그림자 카드 |
| 명세서 배경 | greenbar 줄무늬 | 순수 흰색 |
| 밴드 헤더 | 초록 배경 | 무채색 헤더 + 중립 칩 |
| 금액 폰트 | Consolas 모노 | Paperlogy + tabular-nums |
| 연간 합계 | 먹색 숫자 | 초록 tint 블록 + 초록 강조 |
| "참고" 직인 | 빨간 원형 도장 | **삭제**(HTML·CSS·PNG 모두) |
| 액션 버튼 | 카드 밖 좌측 정렬 | 하단 중앙 액션 바 |

| 시그니처 | 없음 | 월별 지급 타임라인 리본 |

- **HTML 구조 변경**: masthead 축소, 액션 바 위치 이동, 타임라인 마크업 추가, 밴드/직인/perf 요소 제거. `id`(계산 바인딩용)는 **가능한 유지**해 `app.js` 렌더 로직 영향 최소화.
- **app.js**: DOM 렌더 바인딩은 유지. 변경점 = ① 직인 관련 코드 없음(HTML만 제거) ② PNG `drawSlip()`에서 직인·모노폰트·줄무늬 톤을 새 팔레트/Paperlogy 톤으로 갱신 ③ 타임라인 렌더 함수 신설(표시 전용). **calc.js·data/·규정은 손대지 않음.**

## 5. 폰트 조달 (Paperlogy 셀프호스팅)

- OFL 라이선스 → `fonts/` 폴더에 woff2 커밋 합법. 출처: `github.com/fonts-archive/Paperlogy`(또는 눈누).
- 사용 무게만 최소 반입: **Paperlogy-Bold(700)·ExtraBold(800)**(연간 합계 900 쓰면 Black 추가).
- `@font-face`는 상대경로(`url("../fonts/Paperlogy-ExtraBold.woff2")`)로 file:// 호환.
- 파일 조달: 구현 단계에서 curl 다운로드 시도 → 실패 시 사용자가 눈누에서 받아 `fonts/`에 배치(경로는 미리 확정). **폴백 스택 덕분에 파일 부재 시에도 화면 미파손**(Pretendard/Malgun로 대체 렌더).
- OFL 준수: `fonts/`에 라이선스 원문(OFL.txt) 동봉.

## 6. 정확도·검증 (불변 원칙 준수)

- **숫자·규정 불변**: `data/salary-2026.js`, `data/allowance-rules.js`, `js/calc.js` **일절 수정 금지**. 이번은 표현층 리팩토링.
- 수정 후 필수: `node test/verify.mjs`(계산·봉급표 앵커, 표현층 변경이 계산을 안 깼는지) + `test/probe.html`(호봉 즉시 반영·4카드·연합계·경계 13건).
- **표현층 신규 검증(probe에 추가 검토)**:
  - 직인·perf·줄무늬 DOM 부재 확인.
  - 액션 바가 카드 아래 존재·클릭 동작(복사/PNG) 유지.
  - 타임라인 리본 12칸 렌더, 정근(1·7월)·명절(대표월) 마커 위치.
  - 금액 실시간 갱신 시 폭 출렁임 봉쇄(tabular-nums + 우측정렬 고정폭) — **데스크톱 스크린샷으로 육안 확인**(좁은 뷰포트 직접 스크린샷은 이 PC 신뢰도 낮음 → iframe 프로브 병행).
- 접근성 바닥: 반응형(모바일까지), `:focus-visible` 유지, `prefers-reduced-motion` 존중.
